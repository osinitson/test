A1. Subject: API Test- dw.getRelatedFilesFilter()

Feature: DRF

Subfeature: API

Frequency: 4

Smoke: N

Acceptance: N

Server Models: PHP MySQL

Keywords: API

Setup:

  • Download the test folder "DRF" and create a site using it as your local root folder and any server as your remote and testing server.
  • Verify Related Files is enabled (it should be enabled by default). If it's not enabled, then enable it.
  • Copy the api_tests.htm from "DRF/api_test" folder and copy it to your Commands config folder.

Steps:

  1. Open a php file from the DRF site.
  2. Go to Commands > api_tests.
  3. Click on the "Get Current Related Files Filter' button.
  4. Verify that it returns 'No custom filter is set'.
  5. Click the 'Show Only .php' button.
  6. Click the "Get Current Related Files Filter' button again.
  7. Verify that it returns '.php'.
